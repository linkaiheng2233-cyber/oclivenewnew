# REGRESSION — Plugin Manager V2 & Complex Emotion

用于新版插件管理面板（V2）与 Complex Emotion 相关改动的回归检查清单。

## 执行前准备

- 使用最新代码启动应用。
- 先确认 `Ctrl+Shift+F` 在当前环境可触发。
- 测试过程中记录是否需要重启会话/应用才生效。

## 回归清单

### 1) 灰度开关默认关闭

- 进入设置页，确认“启用新版插件管理界面（预览）”默认是关闭状态。
- 按 `Ctrl+Shift+F`，应打开旧版 `PluginManagerPanel`。
- 旧版功能（插件列表、后端模块 Tab、插槽 Tab）正常。

### 2) 开启灰度后路由到 V2

- 设置中打开“启用新版插件管理界面（预览）”。
- 按 `Ctrl+Shift+F`，应打开 `PluginManagerV2`。
- 再次按 `Ctrl+Shift+F` 或按 `Esc`，可正常关闭。

### 3) 左栏分类过滤与角标

- 依次点击：全部 / LLM / Emotion / Complex Emotion / 内置 / 远程 / 目录插件 / 待配置。
- 中栏卡片应随分类即时变化。
- 左栏角标数量应与中栏过滤结果一致。

### 4) 搜索实时过滤

- 在搜索框输入 `llm`、`emotion`、`complex`、`远程` 等关键词。
- 中栏卡片应实时过滤，无需额外提交。
- 清空搜索后应恢复到当前分类的完整列表。

### 5) 卡片选中与右栏详情

- 点击任一卡片后，右栏显示对应标题、说明与模板化配置表单。
- 模板组件渲染正确：`endpoint-config` / `provider-selector` / `slot-selector` / `switch-toggle`。
- 卡片切换时右栏内容同步刷新。

### 6) 配置保存与生效提示

- 修改 LLM 或 Emotion 的配置并点击“应用配置”。
- 应出现成功提示；角色信息中的有效后端值应同步变化。
- 若配置需要重启会话/应用才完全生效，提示语必须明确说明。

### 7) 右栏收起/展开

- 点击右栏“收起”按钮后，详情区折叠为窄栏。
- 再点击“展开”可恢复完整详情面板。

### 8) 一键回到专业版

- 在 V2 顶部点击“一键打开专业版”。
- 应关闭 V2 并打开旧版 `PluginManagerPanel`。
- 再次用 `Ctrl+Shift+F` 可按灰度状态继续切换。

### 9) Complex Emotion 只读状态

- Complex Emotion 卡片展示应为状态型描述（只读），不允许误写环境变量。
- 点击刷新后可更新“是否检测到远程侧车”状态。
- 文案需明确“Complex Emotion 与 emotion 下拉不是同一入口”。
